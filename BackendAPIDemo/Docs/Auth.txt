. Dependencies (Middleware) 🔐

✅ get_current_user() - Token validation
✅ get_current_active_user() - Active user kontrolü
✅ get_current_admin_user() - Admin kontrolü
✅ get_optional_user() - Optional auth

2. Public Endpoints (Auth Gerekmez) 🌐

✅ POST /api/auth/register - Kayıt
✅ POST /api/auth/login - Giriş
✅ GET /api/auth/check-email/{email} - Email kontrolü
✅ GET /api/auth/check-username/{username} - Username kontrolü

3. Protected Endpoints (Auth Gerekir) 🔒

✅ GET /api/auth/me - Mevcut user bilgileri
✅ PUT /api/auth/me - Profil güncelleme
✅ POST /api/auth/change-password - Şifre değiştirme
✅ POST /api/auth/logout - Çıkış
✅ GET /api/auth/profile/{user_id} - User profil görüntüleme

4. Admin Endpoints (Admin Gerekir) 👑

✅ GET /api/auth/admin/users - User listesi
✅ GET /api/auth/admin/statistics - İstatistikler
✅ DELETE /api/auth/admin/users/{user_id} - User silme
✅ PATCH /api/auth/admin/users/{user_id}/status - Status güncelleme

5. Utility 📚

✅ GET /api/auth/endpoints - Endpoint listesi


🎯 KULLANIM ÖRNEKLERİ
1. Register (Kayıt)
bashcurl -X POST "http://localhost:8000/api/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "username": "testuser",
    "password": "SecurePass123",
    "full_name": "Test User"
  }'
Response:
json{
  "success": true,
  "message": "Registration successful",
  "user": {
    "id": "abc-123",
    "email": "test@example.com",
    "username": "testuser",
    "role": "user"
  },
  "token": {
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "token_type": "bearer",
    "expires_in": 86400
  }
}
2. Login (Giriş)
bashcurl -X POST "http://localhost:8000/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "SecurePass123"
  }'
3. Get Current User (Token ile)
bashcurl -X GET "http://localhost:8000/api/auth/me" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
4. Update Profile
bashcurl -X PUT "http://localhost:8000/api/auth/me" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "full_name": "Updated Name",
    "bio": "My new bio"
  }'
